# 英検 準2級プラス　二次試験（面接）ガイド ― 生徒用 ―

> ⚠️ このプリントは2026年6月時点の各種情報をもとにした「概要」です。設問ごとの配点など一部は英検協会が非公開のため、最終確認は[英検公式・試験内容](https://www.eiken.or.jp/eiken/exam/grade_p2plus/solutions.html)と[バーチャル二次試験（入室〜退室をアニメで確認）](https://www.eiken.or.jp/eiken/exam/virtual/grade_p2plus/)で行ってください。

---

## 1. どんな試験？
- **形式**：面接委員（試験官）1人との 1対1 の面接
- **言語**：すべて英語
- **時間**：約 **6〜7分**
- **使うもの**：問題カード（英文 ＋ 3コマのイラスト）

## 2. 当日の流れ
1. 入室 →「面接カード」を渡して着席
2. 名前・受験級の確認、かんたんなあいさつ
3. **問題カード**を受け取る
4. **音読**（20秒だまって読む → 声に出して読む）
5. **質問 No.1〜No.4** に答える
6. カードを返す → 退室

## 3. 問題の中身（音読 ＋ 4問）
| パート | やること | ポイント |
|---|---|---|
| 音読 | 約60語の英文を音読 | 発音・区切り・スピード |
| **No.1** | 英文の内容についての質問 | 答えは**英文の中にある**。`Because …` / `By —ing` で答える |
| **No.2 ナレーション** | 3コマのイラストを説明（**20秒準備**、**カードの1文から始める**） | 過去形・過去進行形、時間の言葉（*Ten minutes later* など）、人物の**気持ち** |
| **No.3** | カードの話題についての**自分の意見** | `Yes / No` → `Because …`（理由を2つ言えると強い） |
| **No.4** | 一般的・社会的な話題についての**自分の意見** | 同上。具体例（*For example, …*）を足す |

> No.3・No.4 は「カードを**裏返して**」答えます（カードは見ません）。

## 4. 評価される 5つの観点
1. **応答内容**（質問に合っているか）
2. **発音**
3. **語彙・文法・語法**
4. **情報量**（どれだけ言えたか）
5. **アティチュード＝態度**（積極的に伝えようとする姿勢／アイコンタクト・声の大きさ）

## 5. 合格の目安
- 合否は **CSEスコア** で判定。二次は **満点625点／合格の目安は約427点**（※各社報告値。要・公式確認）。
- 設問ごとの配点は **非公開**。だからこそ「**黙らずに、まず何か英語で言う**」ことが大切。

## 6. すぐ使えるフレーズ
**🕐 考える時間をかせぐ**
- Let me see… / Well… / That's a good question.

**💬 意見を言う**
- I think (that) … / In my opinion, …
- I have two reasons. First, … Second, …
- For example, …

**🔁 聞き返す（黙らない！）**
- Could you say that again, please?
- Pardon?

**🖼 ナレーションのつなぎ**
- One day, … / Ten minutes later, … / That afternoon, …
- He / She was —ing … / He / She looked happy (worried).

## 7. 合格のコツ
- ❌ 黙る　→　⭕ `Let me see…` で時間をかせぐ
- ❌ 単語だけ　→　⭕ **文**で答える（`Yes, I do. Because …`）
- 大きな声・笑顔・アイコンタクト ＝ **アティチュード点**
- わからなくても、**英語で**聞き返す（だまり込まない）
